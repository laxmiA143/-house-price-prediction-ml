# 🏠 House Price Prediction using Machine Learning

## 📌 Overview
This project predicts house prices based on features like area and number of bedrooms using a machine learning model.

## 🚀 Tech Stack
- Python
- Pandas
- Scikit-learn

## 📊 Features
- Data preprocessing
- Model training using Linear Regression
- Price prediction

## ▶️ How to Run
1. Install required libraries:
   pip install pandas scikit-learn

2. Run the Python file:
   python house_price_prediction.py

## 📈 Output
The model predicts house prices based on input values like area and bedrooms.

## 💡 Learning Outcome
- Understanding of supervised learning
- Basics of regression models
- Working with datasets using Pandas
